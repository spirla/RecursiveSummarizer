TESES JURDICAS:

==========

RESPONSE:

1. O artigo 288 do Código Penal prevê a responsabilização por associacao criminosa. 2. É necessário que a descrição dos fatos seja suficiente para estabelecer o nexo de causalidade. 3. A presença de multiplos atores no cargo de administrador não é suficiente para caracterizar a associação criminosa. 4. A alta rotatividade dos atores não é suficiente para caracterizar a associação criminosa. 5. É necessário que a denúncia individualize a conduta de cada um dos acusados para caracterizar a associação criminosa. 6. A ausência de um vínculo associativo permanente é suficiente para afastar a caracterização da associação criminosa.
